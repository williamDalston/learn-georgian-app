**Part 6:**
Pattern: Root + -eb + Case Ending
- კატები: Root (კატ) + -eb + -ი
- კატებს: Root (კატ) + -eb + -ს
- კატების: Root (კატ) + -eb + -ის

---

**Remember:** The -eb marker always comes between the root and the case ending. Practice this pattern until it becomes automatic!
