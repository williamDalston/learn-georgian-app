# Lesson A2-2: Pluralization and Case Suffixes

**Level:** A2
**Duration:** 45 minutes
**Module:** 2 | **Lesson:** 2
**Estimated Hours:** 0.75 hours

## Learning Objectives
By the end of this lesson, students will be able to:
- Form plural nouns using the -eb suffix
- Apply case endings to plural nouns correctly
- Understand how plurals work with case endings
- Use plural nouns in sentences with proper case agreement

## Key Concepts
- Plural formation with -eb suffix
- Case suffixes with plural nouns
- Root + -eb + case suffix pattern
- Plural declension across cases

## Video Script

### Opening (0:00 - 0:30)
"Gamarjoba! Welcome back! In our last lesson, we mastered the three fundamental cases: Nominative, Dative, and Genitive. Today, we're going to learn how to make nouns plural and how case endings work with plural nouns. This is essential for talking about multiple things - books, cats, children, and more!"

---

### Introduction (0:30 - 3:00)
**[Show comparison: singular vs plural]**

"In English, we usually add -s or -es to make nouns plural: cat → cats, book → books. Georgian has a similar system, but it's a bit different.

In Georgian, we insert the plural marker -eb between the root and the case ending. Let me show you:

- Singular: კატა (kat'a) - cat
- Plural Nominative: კატები (kat'ebi) - cats

Notice the -eb- inserted? And the ending is -ი (i) in the plural Nominative.

But here's the key insight: when we add cases to plural nouns, the pattern is:
**Root + -eb + Case Ending**

Let's learn this step by step!"

---

### Main Content Section 1: Plural Formation Basics (3:00 - 15:00)

**Forming the Plural Nominative**

"The plural Nominative is formed by adding -eb and then -ი (i) to the root.

**[Show transformation chart]**

Singular → Plural Nominative:
- კატა (kat'a) → კატები (kat'ebi) - cats
- წიგნი (ts'igni) → წიგნები (ts'ignebi) - books
- სახლი (sakhli) → სახლები (sakhl-ebi) - houses
- ბავშვი (bavshvi) → ბავშვები (bavshv-ebi) - children

Notice a pattern? For most nouns ending in -ი (i), we replace it with -ები (-ebi).
For nouns ending in -ა (a), we add -ები (-ebi).

**[Show more examples]**

Let's practice identifying singular vs plural:
- წიგნი (ts'igni) - singular
- წიგნები (ts'ignebi) - plural

The -eb- is always the marker, and -ი is the Nominative plural ending."

---

### Main Content Section 2: Plural Dative Case (15:00 - 25:00)

**Plural Dative Formation**

"Now, let's add the Dative case ending to plural nouns. Remember: Root + -eb + Case Ending

**[Show transformation]**

Singular Dative: კატას (kat'as)
Plural Dative: კატებს (kat'ebs)

See? Root (კატ) + -eb + -ს (Dative ending)

**[Show more examples]**

- წიგნები (books - Nominative plural) → წიგნებს (books - Dative plural)
- სახლები (houses - Nominative plural) → სახლებს (houses - Dative plural)
- ბავშვები (children - Nominative plural) → ბავშვებს (children - Dative plural)

**[Show sentence examples]**
მე ვხედავ კატებს [me vkhedav kat'ebs] - I see the cats (plural object in Dative)

მე ვაძლევ ბავშვებს წიგნებს [me vadzlev bavshvebs ts'ignebs] - I give the children books (both in Dative plural)"

---

### Main Content Section 3: Plural Genitive Case (25:00 - 37:00)

**Plural Genitive Formation**

"Now for Genitive plural - showing possession with plural nouns.

**[Show transformation]**

Singular Genitive: კატის (kat'is)
Plural Genitive: კატების (kat'ebis)

The pattern: Root + -eb + -ის (Genitive ending)

**[Show more examples]**

- წიგნები (books - Nominative plural) → წიგნების (books' - Genitive plural)
- სახლები (houses - Nominative plural) → სახლების (houses' - Genitive plural)
- ბავშვები (children - Nominative plural) → ბავშვების (children's - Genitive plural)

**[Show sentence examples]**
ეს არის ბავშვების წიგნები [es aris bavshvebis ts'ignebi] - These are the children's books

სახლების კედლები [sakhlebis k'edlebi] - The walls of the houses"

### Summary (43:00 - 45:00)
"Let's review pluralization:

1. **Plural Formation:** Add -eb between root and case ending
   - Pattern: Root + -eb + Case Ending

2. **Plural Nominative:** Root + -eb + -ი (-ebi)
   - Example: კატები (kat'ebi) - cats

3. **Plural Dative:** Root + -eb + -ს (-ebs)
   - Example: კატებს (kat'ebs) - (to) the cats

4. **Plural Genitive:** Root + -eb + -ის (-ebis)
   - Example: კატების (kat'ebis) - of the cats / cats'

Remember: The -eb- marker always comes between the root and the case ending. This is the key to mastering Georgian plural declension!

In the next lesson, we'll learn about postpositions and how they require specific cases."
