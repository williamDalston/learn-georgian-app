# Public Assets

This directory contains static assets served at the root URL.

## Directory Structure

- `/images` - Image assets (teacher photos, testimonials, hero images, etc.)
- `/icons` - Icon files (SVG, PNG icons)
- `/logos` - Brand logos and "As Seen In" logos

## Asset Guidelines

- Optimize images before adding (compress, proper sizing)
- Use modern formats when possible (WebP, AVIF)
- Name files descriptively (e.g., `teacher-bio-photo.jpg`)
- Keep file sizes reasonable for web performance
- Add alt text descriptions in components using images

## Image Requirements (from plan.md)

### Teacher Photo
- High-quality, professional, warm photograph
- Recommended: 600x600px minimum
- Format: JPG or WebP

### Testimonial Photos
- Real customer photos for credibility
- Recommended: 200x200px (avatar size)
- Format: JPG or WebP

### Hero Image/Video
- Relevant hero imagery or video thumbnail
- Recommended: 1920x1080px (for full-width hero)
- Format: JPG, WebP, or video (MP4, WebM)

### Logos
- "As Seen In" media logos
- Payment method logos (Visa, Mastercard, PayPal)
- SVG or PNG with transparent background preferred
