## Answer Key

**Part 1:**
- -tvis: Genitive
- -shi: Dative
- -dan: Genitive

**Part 2:**
1. ბავშვისთვის
2. სახლში
3. წიგნიდან
4. ქალაქში
5. ქალაქიდან

**Part 3:**
1. ბავშვისთვის
2. სახლში

4. კლასში
5. მასწავლებლისთვის

**Part 4:**
1. მე ვარ სახლში

1. ბავშვის → ბავშვისთვის

3. წიგნის → წიგნიდან

**Part 6:**
1. True
2. False (-shi requires Dative)
3. True
4. False (postpositions come after nouns)
5. False (-shi means 'in' or 'at')
