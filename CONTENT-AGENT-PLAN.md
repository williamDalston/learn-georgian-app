## Content Standards & Format

### Video Script Format
```markdown
# Lesson Title
**Duration:** X minutes
**Level:** A1-A2-B1-B2-C1

## Opening (0:00-0:30)
[Greeting and overview]

## Main Content
### Section 1 (0:30-5:00)
[Detailed script with timing]

## Practice Exercises (5:00-8:00)
[Exercise instructions]

## Summary (8:00-9:00)
[Key takeaways]

## Closing (9:00-9:30)
[Next steps and assignment]
```

### Exercise Format (JSON)
```json
{
  "lessonId": "a1-1",
  "exerciseType": "multiple-choice|fill-blank|matching|audio",
  "questions": [
    {
      "id": "q1",
      "type": "multiple-choice",
      "question": "...",
      "options": [...],
      "correctAnswer": 0,
      "explanation": "..."
    }
  ]
}
```

### Worksheet Format
- PDF template or HTML/CSS template
- Include spaces for student answers
- Answer keys separate

## Agent Coordination Notes

1. **Shared Resources:** All agents should reference:
   - `/components/homepage/course-outline.md` for curriculum details
   - `/lib/data/courseStructure.ts` for lesson structure
   - Common vocabulary database (maintained by Agent 2, used by all)

2. **Consistency Checks:**
   - Use consistent terminology across all lessons
   - Follow the same exercise format
   - Maintain consistent difficulty progression

3. **Dependencies:**
   - Agent 1 must complete A1 before others can reference it
   - Agent 2 needs A1 vocabulary complete
   - Agent 5 needs B1 complete (for Series I reference)
   - Agent 7 needs B2 complete (for verb reference)

4. **Quality Standards:**
   - All scripts: Native speaker review recommended
   - All exercises: Tested for clarity
   - All cultural content: Authentic and accurate
