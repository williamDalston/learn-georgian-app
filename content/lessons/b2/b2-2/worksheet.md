## Answer Key

*(For instructor/self-checking)*

### Part 1:
1. დავწერე, preverb: და-
2. წავიკითხე, preverb: წა-
3. ვნახე, suppletive stem (no simple preverb)
4. წავედი, preverb: წა-
5. მოვედი, preverb: მო-

### Part 2:
- და- = completion/perfective
- წა- = away/perfective
- მო- = toward/perfective
- გა- = perfective
- შე- = perfective

### Part 3:
6. მე დავწერე წერილი
7. ბავშვმა წაიკითხა წიგნი
8. მასწავლებელმა ასწავლა სტუდენტები
9. მან ნახა ფილმი

### Part 4:
10. Subject: მან (3rd person, Ergative), Object marker: მ- (1st person 'me'), Object: წერილი (Absolutive)
11. Subject: მან (3rd person, Ergative), Object marker: გ- (2nd person 'you'), Object: წერილი (Absolutive)
12. დავწერე = 1st person (I wrote), დაწერე = 2nd person (you wrote)

### Part 5:
13. Because 'to run' is intransitive - no object means no case shift
14. to write ✓, to see ✓, to play (if transitive) ✓
