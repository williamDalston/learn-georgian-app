# Lesson A2-4: SVO Sentence Structure Basics

**Level:** A2
**Duration:** 50 minutes
**Module:** 2 | **Lesson:** 4
**Estimated Hours:** 0.83 hours

## Learning Objectives
By the end of this lesson, students will be able to:
- Build simple Subject-Verb-Object (SVO) sentences in Georgian
- Understand basic verb conjugation in Series I (present tense)
- Use proper case endings with verbs
- Form simple sentences with correct word order and case agreement

## Key Concepts
- SVO structure (Subject-Verb-Object)
- Basic verbs (to see, to read, to give)
- Present tense conjugation
- Series I verbs introduction
- Case agreement in sentences

## Video Script

### Opening (0:00 - 0:30)
"Gamarjoba! Now that we've mastered cases and postpositions, we're ready to build complete sentences! Today, we'll learn how to put it all together: Subject-Verb-Object structure with basic verbs. This is where everything clicks - you'll finally be able to express complete thoughts in Georgian!"

---

### Introduction (0:30 - 5:00)
**[Show sentence structure comparison]**

"In English, we say: 'I see the cat.' Subject (I) - Verb (see) - Object (cat).

In Georgian, the structure is similar, but we need to use the correct cases:
მე ვხედავ კატას [me vkhedav kat'as]
- მე [me] - I (subject)
- ვხედავ [vkhedav] - see (verb, Series I present)
- კატას [kat'as] - the cat (object, Dative case)

Notice: The verb has a prefix ვ- (v-) meaning 'I', and the object is in Dative case.

Today we'll learn to build sentences like this!"

---

### Main Content Section 1: Basic Verbs - To See (5:00 - 18:00)

**Verb: ხედვა (khedva) - to see**

"Let's start with a very common verb: to see.

**[Show conjugation]**
მე ვხედავ [me vkhedav] - I see
შენ ხედავ [shen khedav] - You see
ის ხედავს [is khedavs] - He/She sees

Notice the pattern:
- მე (I) → verb has ვ- prefix
- შენ (you) → verb has no prefix
- ის (he/she) → verb has -ს suffix

**[Show sentence examples]**
მე ვხედავ კატას [me vkhedav kat'as] - I see the cat
შენ ხედავ წიგნს [shen khedav ts'igns] - You see the book
ის ხედავს სახლს [is khedavs sakhl-s] - He/She sees the house

**[Key point]**
The object (what is seen) is always in Dative case!"

---

### Main Content Section 2: Basic Verbs - To Read (18:00 - 32:00)

**Verb: კითხვა (kit'khva) - to read**

"Another essential verb: to read.

**[Show conjugation]**
მე ვკითხულობ [me vkit'khulob] - I read
შენ კითხულობ [shen kit'khulob] - You read
ის კითხულობს [is kit'khulobs] - He/She reads

**[Show sentence examples]**
მე ვკითხულობ წიგნს [me vkit'khulob ts'igns] - I read the book
ბავშვი კითხულობს წიგნს [bavshvi kit'khulobs ts'igns] - The child reads the book
ისინი კითხულობენ წიგნებს [isini kit'khuloben ts'ignebs] - They read the books (plural object)

**[Practice]**
Let's build sentences: 'The teacher reads the book' → მასწავლებელი კითხულობს წიგნს"

---

### Main Content Section 3: Basic Verbs - To Give (32:00 - 45:00)

**Verb: მიცემა (mitsema) - to give**

"To give is interesting because it has two objects:
- Direct object (what is given)
- Indirect object (to whom it is given)

Both are in Dative case!

**[Show conjugation]**
მე ვაძლევ [me vadzlev] - I give
შენ აძლევ [shen adzlev] - You give
ის აძლევს [is adzlevs] - He/She gives

**[Show sentence examples]**
მე ვაძლევ ბავშვს წიგნს [me vadzlev bavshvs ts'igns] - I give the child a book
- ბავშვს (bavshvs) - to the child (indirect object, Dative)
- წიგნს (ts'igns) - the book (direct object, Dative)

**[More examples]**
მასწავლებელი აძლევს ბავშვებს წიგნებს [masts'avlebeli adzlevs bavshvebs ts'ignebs] - The teacher gives the children books (plural)"

### Practice Exercise: Building Sentences (45:00 - 48:00)
**[Interactive exercise]**

"Let's build sentences together:

1. I see the house.
   → მე ვხედავ სახლს

2. The child reads a book.
   → ბავშვი კითხულობს წიგნს

3. You give me a book.
   → შენ მაძლევ წიგნს (მე becomes მ- prefix in verb)

**[More practice]**"

### Summary (48:00 - 50:00)
"Today we learned:

1. **SVO Structure:** Subject - Verb - Object
   - Subject in Nominative
   - Verb agrees with subject
   - Object in Dative

2. **Basic Verbs:**
   - ხედვა (to see) - ვხედავ, ხედავ, ხედავს
   - კითხვა (to read) - ვკითხულობ, კითხულობ, კითხულობს
   - მიცემა (to give) - ვაძლევ, აძლევ, აძლევს

3. **Verbs with two objects:** Both take Dative case

In the next lesson, we'll expand our vocabulary with high-frequency words!"
